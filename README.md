# TrePav Whitepaper

一個基於創新算法的綜合數字資產交易平台白皮書網站。

## 📖 關於 TrePav

TrePav 是一個結合先進交易算法與社區驅動激勵的革命性平台。我們致力於民主化數字資產交易，為所有參與者創造公平、透明的交易環境。

## 🌟 主要特性

- **智能交易模型** - 自動定價與流動性保證
- **交易獎勵** - 每筆交易獲得 1.8%-4.1% 獎勵
- **推薦計劃** - 建立網絡並獲得被動收入
- **多資產支持** - 交易加密貨幣、數字資產和收藏品
- **多鏈支持** - 跨多個區塊鏈訪問資產
- **分數 NFT** - 擁有珍貴 NFT 的部分所有權

## 🚀 快速開始

### 本地開發

1. 克隆倉庫
```bash
git clone https://github.com/yourusername/trepav-whitepaper.git
cd trepav-whitepaper
```

2. 使用本地服務器運行（Python）
```bash
python3 -m http.server 8000
```

3. 在瀏覽器中訪問 `http://localhost:8000`

### GitHub Pages 部署

1. 將倉庫推送到 GitHub
2. 在倉庫設定中啟用 GitHub Pages
3. 選擇 `main` 分支作為源
4. 您的網站將在 `https://yourusername.github.io/trepav-whitepaper` 發布
